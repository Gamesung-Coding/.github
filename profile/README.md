## 🚀 About Us

안녕하세요! 저희는 **Gamesung Coding(감성코딩)** 팀입니다.
저희는 기술이 단순히 기능적인 편의를 넘어, **모든 사람이 동등하게 정보를 누릴 수 있는 세상**을 만드는 도구가 되어야 한다고 믿습니다.

디지털 소외 계층(시각 장애인, 고령자 등)을 위한 웹 접근성(Accessibility, A11y) 기술을 실제 서비스에 적용하여 **사회적 가치(Social Impact)**를 실현하는 프로젝트를 진행하고 있습니다.

### 💡 Why A11yMarket? (Social Contribution)

기존의 쇼핑몰은 스크린 리더 사용이나 색각 이상, 저시력 사용자를 위한 배려가 부족했습니다.
저희는 이러한 **정보 불평등(Digital Divide)** 문제를 해결하기 위해 다음과 같은 기술적 도전을 시도했습니다.

1.  **AI 기반 상품 분석:** 시각 장애인이 이미지를 볼 수 없어도, **Google Gemini AI**가 상품 이미지를 분석하여 상세한 텍스트 설명을 제공합니다.
2.  **맞춤형 접근성 UI:** 사용자별(고령자, 저시력 등) 신체 특성에 맞춘 <b>개인화된 UI 설정(글자 크기, 대비, TTS)</b>을 프로필로 제공합니다.
3.  **신뢰 가능한 플랫폼:** **Toss Payments** 연동과 투명한 판매자 등급 시스템으로 안전한 거래 환경을 구축했습니다.
